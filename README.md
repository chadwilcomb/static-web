# static-web
## About
This is a basic template for creating a static single-page web app using React, Webpack (with hot reload), the Ampersand Router and App modules, Stylus, Bootstrap 3, Font Awesome. It also comes with the Google Material Design Color Palette that makes it easy to create/change themes.

Instructions:

1. Git clone
2. Run ```npm install``` to download and install dependencies
3. Run ```npm start``` to start the Webpack Dev Server (with hot reload)
4. To change the primary color scheme, update the ```$color-theme``` variable in _variables.styl
